# Тестирование CryptoMarket Regime Classifier

Этот проект тестирует эффективность подхода HMM + LSTM для определения рыночных режимов.

## Источник

Репозиторий: https://github.com/akash-kumar5/CryptoMarket_Regime_Classifier

## Структура

- `src/` - исходный код проекта
- `dashboard/` - Streamlit интерфейс
- `models/` - обученные модели
- `TEST_PLAN.md` - план тестирования

## Установка

```bash
pip install -r requirements.txt
```

Дополнительные зависимости (из кода):
- talib
- hmmlearn
- tensorflow/keras
- scikit-learn
- numpy
- pandas

## Использование

См. `TEST_PLAN.md` для детального плана тестирования.
